K문서 PDF는 여러 장의 이미지를 한 개의 PDF로 정리하는 간단한 Android 도구입니다.

주요 기능
- JPG, PNG, WebP 이미지 최대 10장 선택
- 원하는 순서로 PDF 페이지 재정렬
- 불필요한 이미지 삭제
- A4 세로/가로와 보통, 좁게, 없음 여백 선택
- 원하는 파일 이름과 위치로 PDF 저장

파일은 기기 안에서 처리됩니다
선택한 이미지와 생성된 PDF를 서버에 업로드하지 않습니다. 앱은 인터넷 연결, 로그인, 광고 또는 분석 기능을 사용하지 않습니다. Android의 시스템 사진 선택기와 저장 화면을 사용하므로 사진이나 저장소 전체 권한을 허용할 필요도 없습니다.

사용 방법
1. PDF로 묶을 이미지를 선택합니다.
2. 페이지 순서를 정리하고 여백을 선택합니다.
3. PDF를 만든 뒤 원하는 위치에 저장합니다.

스캔한 서류, 영수증, 증빙 사진처럼 여러 이미지를 하나의 A4 PDF로 제출해야 할 때 사용할 수 있습니다. 파일은 사용자가 선택한 위치에만 저장되며, 이후 보관과 삭제도 사용자가 직접 관리합니다.
