# K문서툴 Tokens

## Primitive
- Neutral background: `#f7f8fb`
- Surface: `#ffffff`
- Surface muted: `#f4f6fa`
- Ink: `#101828`
- Muted text: `#667085`
- Hairline: `#d0d5dd`
- Strong line: `#18181b`

## Category Accents
- Business: `#155eef`
- PDF: `#e11d48`
- Image: `#d97706`

## Typography
- Body: Pretendard with system Korean fallbacks.
- UI labels: 12-14px, sentence case, no decorative uppercase.
- Card titles: 16-18px, 700-800 weight.
- Page titles: 30-38px, tight but not viewport-scaled.

## Layout
- Page max width: 1120-1180px for utility pages.
- Card radius: 8-12px.
- Inputs/buttons: 44px minimum touch height.
- Card gaps: 20-34px depending on viewport.
- Avoid nested cards and decorative gradients.

## Component Notes
- Tool cards use white surfaces, category-colored borders/icons, and neutral text.
- Search and upload areas are primary actions.
- Upload dropzones must be large, readable, and explicitly local/private.
