# K문서툴

[K문서툴](https://kdoctool.kr/)은 설치와 로그인 없이 브라우저에서 사용하는 무료 문서·PDF·이미지 도구입니다. 선택한 파일과 문서 입력값은 서버로 전송하지 않습니다.

## 바로 사용하기

- [거래명세서 자동작성](https://kdoctool.kr/tools/transaction-statement-generator/)
- [JPG PDF 변환](https://kdoctool.kr/tools/jpg-to-pdf-converter/)
- [사진 용량 줄이기](https://kdoctool.kr/tools/photo-size-reducer/)
- [도장 배경 제거](https://kdoctool.kr/tools/stamp-background-remover/)
- [사진을 1MB 이하로 제출하기](https://kdoctool.kr/problems/photo-under-1mb/)
- [전체 도구](https://kdoctool.kr/tools/)

## 개발

```bash
npm install
npm run dev
```

배포 전 전체 검사는 다음 명령으로 실행합니다.

```bash
npm run build
npm run test:funnel
npm run test:smoke
```
